## API通信勉強用アプリ

### 概要

Retrofit2の勉強成果として作成した簡単なアプリです。<br>
写真を選択してお気に入りにすることができます。<br>

<img src="https://user-images.githubusercontent.com/60771916/147036450-5ab53984-7a35-428d-9a1d-6c4685c8143b.gif" width="400">

### 使用したライブラリ
- Retrofit2
- Glide
- Room
APIはPixabay APIを使用しました。
